# attention

一个用于管理 **学习、生活、提醒与执行** 的个人系统仓库。

## 目标

这个仓库不是普通笔记堆放区，而是一个持续更新的 **个人管理系统**。当前重点是：

- 把课程 assessment、考试、DDL 和提醒节点结构化
- 把“知道要做什么”变成“按时间执行什么”
- 用固定规则减少拖延、模糊判断和临时抱佛脚
- 逐步扩展到学习、生活、休闲三个板块

## 当前范围

目前已整理的内容主要集中在：

1. **学习系统总览**
2. **课程与考试管理**
3. **提醒与执行规则**
4. **近期行动计划**

## 使用方式

这个仓库后续建议按以下方式维护：

- 新增课程信息时，先更新 `docs/academic-tracker.md`
- 新增或修改提醒时，更新 `docs/reminders-and-rules.md`
- 每周调整重点时，更新 `docs/current-action-plan.md`
- 如果管理框架发生变化，再更新 `docs/system-overview.md`

## 管理原则

- 不靠模糊印象管理任务
- 不靠考前突击解决所有课程
- 优先处理“高权重 + 近节点 + 低容错”事项
- 每项任务都尽量转化为 **明确时间 + 明确动作 + 明确结果**
